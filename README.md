## 📖 About

This repository is a curated collection of Jupyter notebooks designed to build a strong foundation in Machine Learning.

Each notebook focuses on a single topic and includes:

* 📚 Conceptual explanations
* 🧠 Mathematical intuition
* 💻 Clean Python implementations
* 📊 Visualizations
* 🎯 Practical examples

Whether you're learning Machine Learning for the first time or revising important topics, these notebooks are structured to make the learning process simple and effective.
